# Low HP Studios - Project Rules & Brand Guidelines

## Brand Identity

### Studio Name: Low HP Studios
- Tech-first with subtle gaming roots (NOT a gaming company)
- "Low HP" represents resilience, building under pressure, smart recovery

### Product: Loadout
- Career Operating System / Resume Engine
- Professional identity management platform

---

## Design Principles

### Color System
- **Primary**: Deep Navy / Near Black (#0a0a0f)
- **Accent**: Electric Cyan (#00e5ff) - use sparingly (<25%)
- **Dark mode first** - always design for dark backgrounds
- Flat colors only - NO gradients, NO glows

### Typography
- Clean, geometric sans-serif
- Angular but slightly rounded
- Modern and developer-friendly

### Visual Style
- Minimal and flat
- High contrast
- Rounded edges for approachability
- Subtle identity hints over literal gaming metaphors
- Professional over playful

---

## Code Conventions

### Stack
- Next.js 14+ with App Router
- TypeScript (strict mode)
- Tailwind CSS for styling
- Framer Motion for animations

### File Structure
```
app/           → Next.js app router pages
components/    → Reusable React components
lib/           → Utility functions
public/        → Static assets
```

### Component Guidelines
- Use functional components with TypeScript
- Prefer composition over inheritance
- Keep components focused and single-purpose
- Use Framer Motion for animations

---

## Brand Voice

- Professional core, playful edge
- Developer-centric language
- Startup energy without being corporate
- Confident but not arrogant
- Avoid buzzwords and fluff
